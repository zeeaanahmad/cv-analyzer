# CV Analyzer

AI-powered CV analysis tool that scores your resume against a job description 
and gives actionable feedback.

**Live demo:** https://cv-analyzer-8m2l.onrender.com

## Built with
- Python / Flask
- Google Gemini API
- PyMuPDF (PDF text extraction)
- Deployed on Render

## Features
- Upload CV as PDF
- Paste any job description
- Get a match score, strengths, gaps, and improvement suggestions
